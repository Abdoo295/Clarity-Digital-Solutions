# Clarity - Digital Solutions 🚀

A modern responsive digital agency website built to showcase digital services and solutions with a clean, professional, and user-friendly design.

## 📌 Overview

Clarity is a responsive landing page for a digital solutions company.  
The project focuses on creating a modern UI with smooth layouts, responsive behavior, and interactive components.

## ✨ Features

- Fully Responsive Design 📱💻
- Modern Hero Section
- Responsive Navbar with Mega Menu
- Bootstrap Accordion FAQ Section
- Interactive Buttons & Hover Effects
- Contact Form UI
- Custom Styled Components
- Clean and Organized Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome Icons

## 🎨 Design Highlights

- Custom gradients
- Box shadows
- Smooth hover animations
- Modern card designs
- Mobile-first responsive approach
